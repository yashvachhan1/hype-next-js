The All-in-One Growth Suite for WooCommerce. Wholesale Pricing + Premium Shop Filters in one free plugin.

== Description ==

**Woo Custom Suite** is the only plugin you need to turn a standard WooCommerce store into a professional B2B & B2C Powerhouse. 

We believe essential features shouldn't be expensive. That's why we include **Wholesale Pricing**, **Quantity Discounts**, and **Premium AJAX Filters** all for free.

### 🚀 Core Features (Free Forever)

*   **Wholesale Pricing**: Set percentage discounts globally for specific user roles.
*   **Premium Shop Design**: Instantly transforms your shop page into a modern, purple-themed grid with "Inter" typography.
*   **AJAX Filtering**: Filter products by price, category, and attributes without reloading the page.
*   **Role Management**: Configure "Category 1" and "Category 2" customers easily.

### 💎 Pro Features (Included FREE for Early Adopters)

> **Limited Time Offer**: All features below are currently unlocked for FREE as part of our "Growth First" launch strategy.

*   **Tiered Pricing**: "Buy 5, Get 5% Off" logic.
*   **Tax Exemption**: Automatically remove tax for approved wholesale partners.
*   **One-Click Registration**: Shortcode to allow users to apply for wholesale status.
*   **Dynamic Brand Logos**: Automatically display brand logos in the sidebar.

== Installation ==

1. Upload the `woo-custom-suite` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Woo Suite** in the left admin menu to configure your settings.
4. To use the Shop Filter, create a new page and add the shortcode `[woo_custom_suite_shop]`.
